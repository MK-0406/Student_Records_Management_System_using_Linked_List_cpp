# Student Records Management System (Linked List - C++)

Simple console application implementing a student records management system using singly linked lists in C++.

## Overview

This project stores and manages student records, exam results and subjects using linked-list data structures. It demonstrates basic OOP in C++ across classes such as `Student`, `Exam`, `Subject`, `List`, and `Node`.

## Features

- Add / remove / search student records
- Manage exam and subject entries
- Simple file-based sample data for testing

## Project files

- `app.cpp` — application entrypoint and main menu
- `Student.cpp`, `Student.h` — `Student` class and related operations
- `Exam.cpp`, `Exam.h` — `Exam` class and parsing utilities
- `Subject.cpp`, `Subject.h` — `Subject` class
- `List.cpp`, `List.h` — linked list implementation and helpers
- `Node.cpp`, `Node.h` — linked list node definition
- sample data files (e.g. `student.txt`, `exam.txt`, `student_result.txt`, `exam (1a) - ... .txt`)

## Build & Run (Visual Studio 2025 / 2026)

Follow these steps to clone this repository from GitHub and run it in Visual Studio 2025 or 2026.

1. Clone the repo locally:

```powershell
git clone https://github.com/MK-0406/Student_Records_Management_System_using_Linked_List_cpp.git
cd Restaurant_Ordering_System_cpp
```

2. Open the project in Visual Studio:
- In Visual Studio: **File → Open → Folder...** and select the cloned repository folder.

3. Build and run:
- Build: **Build → Build Solution** (or `Ctrl+Shift+B`).
- Run with debugger: **Debug → Start Debugging** (or `F5`).
- Run without debugger: **Debug → Start Without Debugging** (or `Ctrl+F5`).
